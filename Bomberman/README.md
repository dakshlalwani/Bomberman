# Bomberman

## How to run game

1). Go to the directory 'Bomberman'

2). Execute 'python bomberman.py' or 'python3 bomberman.py'

3). Game will start with the following cantrols:

    -> key 'w' = move up

    -> key 'a' = move left

    -> key 'd' = move right

    -> key 's' = move down

    -> key 'b' = drop bomb

    -> key 'q' = quit the game

4). score info:

    kill an enemy -> score += 100

    break a brick -> score += 20
